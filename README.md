# Feel the Movies Dashboard 2.0

This is the dashboard of Feel the Movies project.

### Install:

The dashboard depends on back-end project. This project uses Create React App, so inside the folder just run:

```sh
yarn install
```

### Run:

```sh
yarn run start
```
